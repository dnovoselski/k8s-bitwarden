# 🔐 Bitwarden on Kubernetes with Helm

Deploy Bitwarden (Vaultwarden) — a self-hosted password manager — on a Kubernetes cluster using Helm. This setup is production-ready with persistent storage, Ingress with TLS, and namespace isolation.
